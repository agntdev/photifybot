# PhotifyBot

Telegram bot for on-demand AI image generation — single or 4-variant outputs, selectable sizes, regenerate/upscale controls; 10 free trials, paid packs via external link.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
